# 基于Spring Cloud的学生选课和教学评估系统 - 技术栈与实现方案

## 项目概述

### 主要内容
设计和实现一个基于微服务架构的在线选课与教学评估系统，主要包括管理员、教师和学生三个角色：

- **管理员功能**：系统用户管理、课程信息管理、选课管理、教学评估模板管理、系统数据统计与分析等功能
- **教师功能**：我的课程管理、查看选课学生名单、学生成绩录入与管理、查看教学评估结果与统计报告等功能
- **学生功能**：在线选课与退课、个人课表查看、成绩查询、参与匿名教学评估等功能

### 主要目标
开发一款功能全面的在线选课与教学评估系统，实现一个功能完整、用户体验良好的在线选课与评估平台，并提供科学、匿名的教学评估通道，提升教学管理效率。

### 技术要求
1. 采用Java进行系统开发
2. 采用SpringCloud框架搭建各个独立的微服务
3. 采用Web技术进行前端开发
4. 采用MySql数据库进行数据存储和操作
5. 要求熟悉软件开发流程，掌握代码规范、软件测试等软件工程实践
6. 完成开题报告和毕业设计说明书，要求表述通顺、用词准确、逻辑清晰、图表规范、格式符合学校要求的写作标准

## 微服务架构设计

基于毕设需求，建议的微服务划分如下：

### 1. cloud-gateway（已存在）
**功能**：API网关服务
- 路由转发与负载均衡
- 统一入口管理
- 请求过滤与安全控制

### 2. cloud-auth（新增）
**功能**：用户认证授权服务
- 用户登录登出
- JWT令牌生成与验证
- 权限验证与授权

### 3. cloud-user（已存在，需扩展）
**功能**：用户管理服务
- 管理员、教师、学生信息管理
- 用户CRUD操作
- 角色权限管理
- 用户信息查询

### 4. cloud-course（新增，替换cloud-order）
**功能**：课程管理服务
- 课程信息管理
- 课程分类管理
- 课程发布与下架
- 课程详情查询

### 5. cloud-selection（新增）
**功能**：选课管理服务
- 学生选课功能
- 选课冲突检测
- 退课功能
- 选课记录查询
- 个人课表生成

### 6. cloud-evaluation（新增）
**功能**：教学评估服务
- 评估模板管理
- 学生匿名评估
- 评估结果统计
- 评估报告生成

### 7. cloud-grade（新增）
**功能**：成绩管理服务
- 教师录入成绩
- 成绩修改与删除
- 学生成绩查询
- 成绩统计分析

### 8. fegin-api（已存在，需扩展）
**功能**：服务间调用接口定义
- 公共实体类
- Feign客户端接口
- 统一返回结果封装
- 异常处理类
